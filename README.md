# sinh

[![Build Status][travis-svg]][travis]

Calculate the hyperbolic sine of a number.

## Example

``` javascript
var sinh = require('sinh');

sinh(1);
// => 1.1752011936438014
```

## Installation

``` bash
$ npm install sinh
```

## API

``` javascript
var sinh = require('sinh');
```

### `sinh(x)`

Given a _Number_ `x`, returns the hyperbolic sine of `x` as a _Number_`.
